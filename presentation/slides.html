Architecture Overview

Our solution is built on a shared VPC with integrated services that work together. Cryptpad serves as our front-end collaborative platform, MinIO provides the underlying storage infrastructure, and Grafana monitors everything, pulling metrics from CloudWatch.

Cryptpad
Functionality and Features
  • Real-time collaborative document editing with end-to-end encryption
  • Supports rich text documents, kanban boards, code editor and more
  • Zero-knowledge architecture ensuring privacy and security
  • No user accounts required for basic collaboration
  • Content remains accessible through unique document URLs
AWS Implementation
  • EC2 Instance: t3.small instance provides sufficient CPU and memory for Node.js
  • EBS Volume: 20GB persistent storage safeguarding documents if instance terminates
  • S3 Backup: Automated daily backups to S3 for disaster recovery
  • Security Groups: Controls access to specific service ports (3000, 3001, 3003)
Server Deployment
  • Install required dependencies, then clone and set up CryptPad from its official repository
  • Customize the configuration file to define server origin, ports, storage paths, and instance behavior
  • Schedule cron jobs to automatically clean up expired and archived documents for efficient storage management
  • Use Amazon S3 to back up key directories such as datastore/, data/, and config.js for disaster recovery
  • Set up a systemd service to ensure CryptPad automatically restarts after system reboots
MinIO
Functionality and Features
  • Self-hosted solution maintaining control of stored data
  • Intuitive web console for easy bucket creation and file management
  • Secure access control with API keys and policy-based permissions
  • Built-in data protection with versioning, object locking and replication
  • S3-compatible API enabling familiar operations and seamless integration with AWS CLI, SDKs, and third-party tools
AWS Implementation
  • Multi-AZ Deployment: Servers in two zones prevent downtime during outages
  • Load Balancer: Routes traffic efficiently between instances for consistent access
  • Docker Containers: Packages software for easy deployment and updates
  • EBS Storage: 30GB per server (60GB total) with data that persists beyond instance lifecycle
  • CloudWatch Metrics: Monitors performance with custom dashboards tracking critical indicators
Grafana
Functionality and Features
  • Comprehensive dashboards visualizing system and application metrics
  • Real-time monitoring of all infrastructure components
  • Pre-configured dashboards for MinIO and Cryptpad
  • Alert capabilities for performance thresholds
  • Single pane of glass for operational monitoring
AWS Implementation
  • EC2 Instance: t3.micro instance sufficient for monitoring requirements
  • CloudWatch Integration: Pulling metrics from all services
  • S3 Storage: Persistent configuration and dashboard backups
  • Security Groups: Controlled access to management interface
  • Custom Dashboards: Purpose-built visualizations for our specific services
